Bootstrap 5 Cheatsheet

Grid System

  • Bootstrap's grid system can adapt across all six default breakpoints, and any breakpoints you customize. Breakpoints are based on min-width media queries
  • Rows are wrappers for columns. ("row", "col")
  • Columns are flexible. There are 12 template columns available per row. Column classes indicate the number of template columns to span (e.g., col-4 spans four).

Breakpoints

xs: 0, sm: 576px, md: 768px, lg: 992px, xl: 1200px, xxl: 1400px

Variable width content

Use col-{breakpoint}-size classes to size columns based on the natural width of their content.

col-sm-6 col-lg-4 col-xxl-3
  • If the screen size is at least sm, then column width: 50% (6/12)
  • If the screen size is at least lg, then column width: 33% (4/12)
  • If the screen size is at least xxl, then column width: 25% (3/12)
The smaller screen size, the larger the percentage should be (think of viewing thumbs on mobile)

Row columns

Use the responsive .row-cols-{breakpoint}-numberOfColumns classes to quickly set the number of columns that best render your content and layout. Row columns classes are set on the parent .row as a shortcut

Bootstrap comes with three different containers:

  • container, which sets a max-width at each responsive breakpoint
  • container-{breakpoint}, which is width: 100% until the specified breakpoint
  • container-fluid, which is width: 100% at all breakpoints
